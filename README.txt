Реализованная функциональность:
1. Эмуляция данных сотрудников
2. Анализ эмулированных данных для выявления риска возможного выгорания
3. Онлайн анкета для углубленной проверки, при выявлении повышенного риска выгорания сотрудника

Особенность проекта в следующем:
1.Использование Docker для удобства развертывания проекта на любом устройстве
2.Готовая среда для запуска
3.Использование психологического теста, для полного выявления риска выгорания
 
Основной стек технологий:
MySQL, Docker, HTML, CSS, Git

Среда запуска:
1.
  - Windows 11
  - Windows 10 (64-битная версия Home, Pro, Enterprise, и Education)
  - Windows Server 2019
  - Windows Server 2016
  - macOS Mojave 10.14 или выше
  - macOS High Sierra 10.13 или выше
  - Любая Linux-система с поддержкой контрольных групп в ядре
2. Браузер, для корректной работы сайта

Установка:

Чтобы открыть Docker проект на другом компьютере, необходимо выполнить следующие шаги:

1. Убедитесь, что на компьютере установлен Docker. Если не установлен, нужно скачать и установить Docker Desktop для Windows или macOS, либо Docker Engine для Linux.
2. Затем необходимо склонировать репозиторий с проектом с помощью системы контроля версий, такой как Git.
3. После этого запустите терминал или командную строку и перейдите в каталог с проектом.
4. Используйте команду docker-compose up для запуска контейнеров, описанных в файле docker-compose.yml, если таковой имеется.

Таким образом, при наличии Docker и проекта можно достаточно легко развернуть его на другом компьютере.
